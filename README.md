这是使用Python3开发的一个小功能，用于获取文件的MD5。
一般的应用场景就是用来对比两个文件的MD5是否一致，用来确保手上的文件与原始文件是否为同一个文件。
本身代码文件很小才2KB，加了UI界面以及打包成exe，集成了Python3运行环境。
所以成品的exe文件有8M大小。
